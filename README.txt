====== INFORMACIÓN ======
En la carpeta de 'Base de Datos' se añade el documento .sql en el que se ha creado toda la estructura de la base de datos de la aplicación.

La carpeta 'sitio web' incluye el prototipo de una página web diseñada para el hospital ficticio al que va dirigida la aplicación.

La carpeta 'Ejecutable' incluye un archivo .jar listo para ejecutar el programa.
IMPORTANTE: No borrar la carpeta 'lib' ya que sin ella no funciona el archivo .jar
